# NaijaFoodPriceAI

**NaijaFoodPriceAI** is a machine learning-powered web application that predicts food prices across Nigerian markets. Using historical market data, vendor details, product categories, packaging information, and location context, the platform provides reliable price estimates to support smarter purchasing and business decisions.

This project was developed as part of the **Nithub AI Ecosystem**, where I contribute as a volunteer building practical AI solutions that create real value for local communities and the Nigerian economy.

---

## Overview

Food prices in Nigeria often change due to seasonal trends, transportation costs, supply chain disruptions, vendor differences, and regional market conditions. These fluctuations can make it difficult for consumers, traders, and researchers to make informed decisions.

**NaijaFoodPriceAI** addresses this challenge by applying machine learning to historical food price data, helping users estimate likely prices based on market, vendor, product, and packaging features.

---

## Purpose and Impact

This project is designed to support different groups of users:

- **Consumers** can plan purchases better and compare likely prices across markets.
- **Vendors and retailers** can make better pricing and stocking decisions.
- **Researchers and policymakers** can study food price patterns for planning and analysis.

By turning historical pricing data into actionable insights, the application helps users make better decisions, reduce uncertainty, and respond more effectively to market changes.

---

## Key Features

- **Food Price Prediction**  
  Predicts food prices using trained machine learning models based on historical data.

- **Location-Aware Input**  
  Supports market-specific prediction using location and contextual information.

- **Fast Web Interface**  
  Built with Flask to provide quick predictions through an easy-to-use interface.

- **Detailed Prediction Context**  
  Includes relevant features such as vendor type, store visited, food category, item type, package type, and date.

- **Responsive Frontend**  
  Designed with HTML, CSS, and JavaScript for a modern user experience across devices.

## Project Workflow

The development of this project involved the following steps:

1. **Data Collection and Cleaning**  
   Gathered and cleaned historical food price data from Nigerian markets.

2. **Exploratory Data Analysis**  
   Studied patterns, trends, and relationships in the dataset to understand key price drivers.

3. **Feature Engineering**  
   Processed and transformed variables such as vendor details, item categories, packaging, and market information for model training.

4. **Model Development**  
   Built supervised machine learning models to predict food prices accurately.

5. **Model Serialization**  
   Saved trained models using **Joblib** for efficient reuse and deployment.

6. **Web App Development**  
   Developed a Flask-based application with dynamic input forms and interactive result pages.

7. **Frontend Design**  
   Created a clean and responsive user interface for accessibility and usability.
